# LearningChinese

欢迎来到中文自学教程！Welcome to the Chinese Self-Study Tutorial!

This repository contains a comprehensive 1-year Mandarin Chinese self-study guide designed for beginners around the world.

## 📖 [Start Learning → chinese_tutorial.md](./chinese_tutorial.md)

### What's Included

- 🎵 Complete Pinyin system (initials 声母, finals 韵母, tones 声调)
- 📅 12 months of structured lessons
- 💬 Real-life dialogues for every scenario
- 📚 Vocabulary lists, grammar points, and exercises
- 🌏 Chinese culture and customs

### Course Overview

| Month | Theme |
|-------|-------|
| 1–2 | Foundations: Pinyin, tones, greetings, numbers |
| 3–4 | Daily life: Family, food, restaurants |
| 5–6 | Getting around: Transport, directions, weather |
| 7–8 | Social life: Work, shopping, technology |
| 9–10 | Health and travel |
| 11–12 | Culture, festivals, and full review |
